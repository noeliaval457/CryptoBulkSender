## 🆘 Troubleshooting Common Issues

If you experience problems, check these items first.

- Connection Errors: If the application cannot connect to the network, check your internet settings. Refresh the connection in the Settings tab.
- Transaction Failures: Ensure your source wallet has sufficient funds to cover both the transfer amount and the transaction fee.
- File Errors: Verify that your CSV file uses the correct format. The address must be in the first column and the amount in the second column.
- Application Hangs: Restart the application if it becomes unresponsive. If the problem continues, restart your computer.

## 📝 Frequently Asked Questions

What blockchain networks does this support?
The app supports EVM-based chains including Ethereum, BSC, and ZKSync. It also supports Solana and Tron. 

Can I export my transaction history?
Yes. Navigate to the "History" tab and click the "Export CSV" button to save a record of your transfers.

Does the software store my private keys?
The software keeps your sensitive data in a local, encrypted vault on your computer. It does not send this information to any remote server.

Is there a limit to how many transfers I can process?
There is no hard limit on the number of transfers. Larger files might require more time to process. Use smaller batches if your list contains thousands of addresses.

How do I clear my history?
Go to the "Settings" menu and select "Clear Cache and Logs." This removes local records of past transactions.
